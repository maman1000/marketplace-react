# Marketplace React

Project React sederhana yang dibuat menggunakan Vite.

## Fitur
- Navbar
- Hero Section
- Categories
- Featured Product
- Trending Product
- Footer
- Fitur authentication (register & login)
- Halaman untuk menampilkan daftar user
- Halaman detail user

## Teknologi
- React
- Vite
- Tailwind CSS

## Cara Menjalankan

```bash
npm install
npm run dev
```
